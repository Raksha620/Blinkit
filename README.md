# Blinkit
The Blinkit Sales Dashboard is a Power BI report providing insights into sales performance for Blinkit, India's last-minute grocery app. This dashboard visualizes key metrics and trends to support data-driven decision-making.

Features
Total Sales: Displays total sales, average sales per transaction, number of items sold, and average customer rating.
Sales Trends: Line chart showing sales trends over time.
Sales Breakdown: Analysis of sales by item type, outlet size, and location type.
Outlet Performance: Table comparing different outlet types on key metrics.
Fat Content Analysis: Sales comparison based on fat content.
Tools
Power BI: For creating the interactive dashboard
